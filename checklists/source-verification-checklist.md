# 자료 검증 체크리스트

## 출처 확인

자료의 작성자, 기관, 발행일을 확인했는가.

최신 정보가 필요한 주제인지 판단했는가.

공식 문서, 논문, 판례, 보도자료, 블로그를 구분했는가.

## 내용 확인

자료가 실제 질문과 직접 관련되어 있는가.

인용한 문장이 원래 맥락과 어긋나지 않는가.

여러 출처가 서로 충돌할 때 어느 쪽이 더 신뢰할 만한지 판단했는가.

## 위험 신호

출처가 오래되었는데 현재 사실처럼 쓰고 있지 않은가.

2차 출처만 보고 원문을 확인하지 않은 것은 아닌가.

숫자, 날짜, 이름, 법령, 판례번호에 오타가 없는가.
