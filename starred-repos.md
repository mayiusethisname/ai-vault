# Starred Repos

GitHub에서 스타해둔 레포 중 실제 작업에 쓸 만한 것을 정리하는 파일이다.

## Stars 원본

- GitHub Stars 페이지: https://github.com/mayiusethisname?tab=stars

현재 이 저장소의 GitHub 연결 도구만으로는 Star 목록을 직접 조회할 수 없다. Star 목록을 정리하려면 Stars 페이지가 공개 웹에서 읽히거나, 사용자가 레포 링크 목록을 붙여넣거나, GitHub API/CLI로 목록을 추출한 뒤 이 파일에 반영한다.

## 분류 기준

### AI 프롬프트 / 에이전트 규칙

ChatGPT, Codex, Claude, Cursor, AGENTS.md, system prompt, workflow prompt와 관련된 레포를 모은다.

### 문서 / PDF / 변환

PDF 변환, 문서 편집, OCR, DOCX, 한글 문서 작업과 관련된 레포를 모은다.

### 법률 / 리서치 / 판례 정리

법률 문서 분석, 쟁점 정리, 검색, 인용, 판례 리서치에 도움이 되는 레포를 모은다.

### 학습 / 연구 / 논문 정리

논문 읽기, 실험 보고서, 전공 개념 설명, 리서치 노트 정리에 도움이 되는 레포를 모은다.

### 개발 자동화 / GitHub 워크플로우

CLI, GitHub Actions, repo 관리, 자동화 스크립트, 코드리뷰 보조 도구를 모은다.

### 보류 / 나중에 확인

좋아 보여서 스타했지만 아직 용도나 품질을 판단하지 못한 레포를 임시로 모은다.

## 작성 형식

```md
## owner/repo-name

분류:

용도:

쓸 상황:

중요 파일:

주의할 점:

내 작업에 적용할 방식:
```

## 후보 목록

아직 등록된 레포가 없다.

새 레포를 추가할 때는 단순 링크만 붙이지 말고, 내가 왜 스타했는지까지 적는다.
