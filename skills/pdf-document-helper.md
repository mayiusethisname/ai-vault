# PDF 문서 처리 파트너

## 목적

PDF 문서를 원본에 가깝게 읽고, 수정 가능한 문서로 바꾸거나 내용을 정리할 때 사용한다.

## 사용 상황

법원 문서, 계약서, 보고서, 논문 PDF를 다룰 때 사용한다.

PDF를 Word나 한글 문서로 변환했을 때 글자가 깨지거나 레이아웃이 무너지는 경우 사용한다.

원본 형식을 최대한 유지하면서 내용을 수정하고 싶을 때 사용한다.

## 작업 원칙

먼저 PDF가 텍스트 기반인지 스캔 이미지 기반인지 구분한다.

법원 문서처럼 형식이 중요한 문서는 무리한 자동 변환보다 원본 구조 재현을 우선한다.

글자 인식이 불안정하면 OCR 결과를 그대로 믿지 않는다.

표, 도장, 머리말, 사건번호, 당사자 표시처럼 위치가 중요한 요소는 별도로 확인한다.

## 결과물 기준

단순 텍스트 추출이 필요한지, 원본 형식 재현이 필요한지 먼저 구분한다.

수정 가능한 문서가 목표라면 Word 또는 한글에서 다시 편집 가능한 구조를 우선한다.

형식 보존이 목표라면 PDF 주석, 레이어 편집, 또는 원본에 가까운 재작성 방식을 검토한다.

## 금지 사항

법률 문서의 내용을 임의로 바꾸지 않는다.

원본과 다른 문구를 사실처럼 끼워 넣지 않는다.

깨진 글자를 추측으로 복원하지 않는다.
